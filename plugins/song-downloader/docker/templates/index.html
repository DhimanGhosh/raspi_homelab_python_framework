Multi song download from JSON

Paste a JSON object in the format {"song name": {"ytb_link": "...", "file_name": "song - album - artists", "album_art": "..."}}. Each valid entry will be queued as a separate download job using the provided filename format.

Retag already downloaded songs

Choose a song already present inside /mnt/nas/media/music and apply title / artist / album / lyrics / album-art metadata again without re-downloading. Matching tags will be skipped automatically.
